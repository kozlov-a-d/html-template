Forms structure

Defualt

Самая обычная форма, все элементы находятся один под другим

  • Неправильный формат. Пример: 88008008080

Line

Модификатор --line

Все item'ы формы располагаются в линию, как inline-block'и.
Также тут добавлен модификатор для кнопки --width-100, чтобы кнопка занимала всё доступное ей пространство.

  • Неправильный формат. Пример: example@domain.ru

Column 1-2 / 1-1-1

Модификатор --columns

Добавляются колонки, внутри которых вставляются item'ы формы

  • Неправильный формат. Пример: example@domain.ru

Пример формы с колонками 1-1-1

  • Неправильный формат. Пример: example@domain.ru

Horizontal

Модификатор --horizontal

У каждого item'а название и поле расположены в линию

  • Неправильный формат. Пример: example@domain.ru

Horizontal + description

Модификатор --horizontal-note

Аналогично horizontal, только к полям добавляется описание
