Grid system

Get Started

Сетку можно использовать 2-мя способами:

  • прописывая классы непосредственно в html (аналогично bootstrap'у)
  • использую mixin'ы в стилях

Inline-вариант

Класс для контейнера grid, элементы обозначаются классами col-X-Y. Ширина каждого блока равна отношению X / Y * 100% - Z , где Z - расстояние между колонками
При необходимости можно вынести часть элемнтов в отдельную строку, добавив им обёртку row.

col-1-6
col-5-6
col-1-4
col-3-4
col-1-2
col-1-2
col-1-1

                         
<div class="grid">


                         
    <div class="col-1-6">col-1-6</div>


                         
    <div class="col-5-6">col-5-6</div>


                         
    <div class="col-1-4">col-1-4</div>


                         
    <div class="col-3-4">col-3-4</div>


                         
    <div class="col-1-1">col-1-1</div>


                         
    <div class="col-1-2">col-1-2</div>


                         
    <div class="col-1-2">col-1-2</div>


                         
</div>

*Класс col в данном случае нужен только в декоративных целях

Default Media Queries

Для адаптивных сайтов вы можете контролировать то, как сетка будет вести себя на определённых разрешения путём добавления специальных классов. Ниже приведена таблица со стандартными значениями:

Key CSS Media Query Разрешение Класс
xs @media (min-width: 320px) ≥ 320px col-xs-*-*
sm @media (min-width: 568px) ≥ 568px col-sm-*-*
md @media (min-width: 768px) ≥ 768px col-md-*-*
lg @media (min-width: 1024px) ≥ 1024px col-lg-*-*
xl @media (min-width: 1280px) ≥ 1280px col-xl-*-*

Эти значения можно переопределить в основном файле стилей \assets\styles\common.css
